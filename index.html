Welcome to Paint

Paint is CSS framework that let's you 'paint' on styles with easy to remember classes. It speeds up development time, makes code easy to maintain, and is totally modular.

Adapt the existing settings to your own needs. Use what you want, scrap the rest. Aside from the basic building blocks: banner, container, module ... nothing else carries any dependencies. If you need to make changes to specific elements, just give that element an id and add it to the overrides section of the css file.

Except for font family and color choices, no extra css was written to create this page. What a time saver, eh?

Neat, I'm vertically centered!

If you wanted make the modules in this row aligned to the top, you would simply add the .align-top class to the modules in this row. You could also apply the .align-bottom class. Middle vertical alignment is the default.

Notice that margins between text in this module are created with the margin-bottom class. This is much more convenient than specifically identifying these elements in your css file and applying the margin that way, or worse, using inline styles.

Use panels for special backgrounds

This is a panel module. Panels are used when you have elements that require colored or special backgrounds.

Left side full width panel

But still in the container!

Normally, if you wanted a full width background color, you would use an id on the section in question. If you wanted to split the screen, with two different colored backgrounds, you will do it a little differently.

Instead of using the container class inside your section, you'll use the container- extended class. That will ensure your modules are able to span the full width of the screen.
